Hooks

Available hooks through Nitro

HookArgumentsDescription
applicationinsights:configconfig:TNitroAppInsightsConfigWhen before starting applicationinsights
applicationinsights:context:tagsclient:TelemetryClient, tags: Record<string, string>, ctx: { event: H3Eevent}Called when all basic tags has been assigned to the telemetry context
applicationinsights:trackRequest:beforeevent: H3Event, trackInfo: RequestTelemetry & IdentifiedCalled before tracking the event.
applicationinsights:trackError:beforeexceptionTelemetry: Contracts.ExceptionTelemetry, ctx: CapturedContextErrorCalled before tracking the event. You can set ctx.shouldTrack to false to prevent the error being tracked.
